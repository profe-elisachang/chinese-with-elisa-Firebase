# AI 生成识字课程 Markdown 格式指令

## 📋 任务说明

请将提供的识字课程教材图片（2-4张）解析并转换为标准 Markdown 格式。图片中通常包含以下内容：
- **字表**：生字列表（汉字、拼音、英文释义）
- **识字句**：包含本课生字的完整句子
- **词组**：相关词组（中文、拼音、英文）
- **语法**：语法规则说明
- **练习**：填空题、选择题等练习题

---

## 🎯 输出格式要求

### 重要提示
- **必须使用阿拉伯数字**：标题格式为 `## 识字9 - 字表`（不是"识字九"）
- **所有表格必须有开头的 `|`**：例如 `| 字 | 拼音 | 英文 | notes |`
- **标题与内容之间必须有空行**
- **严格按照以下格式输出，不要自行修改格式**

---

## 📝 格式规范

### 1. 字表（Vocabulary Table）

**标题格式**（二选一）：
- `## 识字9 - 字表`（推荐，带课数）
- `## 字表`（简化格式）

**表格格式**：
```markdown
## 识字9 - 字表

| 字 | 拼音 | 英文 | notes |
|----|----|----|----|
| 林 | lín | forest, (a surname) | |
| 叔 | shū | uncle | 叔叔 |
| 家 | jiā | family, home, specialist | |
| 口 | kǒu | mouth | 人口 |
```

**字段说明**：
- `字`：汉字（必填）
- `拼音`：拼音（必填，带声调）
- `英文`：英文释义（必填）
- `notes`：可选，通常是该字组成的词组或例句

**注意**：
- 表格第一行必须有开头的 `|`
- 如果某个字段为空，保留空字符串 `|`，不要省略
- 不要添加序号列（如 `#` 列）

---

### 2. 识字句（Literacy Sentence）

**标题格式**（二选一）：
- `## 识字9 - 识字句`（推荐）
- `## 识字句`（简化格式）

**内容格式**：
```markdown
## 识字9 - 识字句

林叔叔家人口真多，有爸爸、妈妈、哥哥、嫂子、弟弟和妹妹。另外，他祖父母呢，以前是军队干部，已经退休很长时间了。
```

**注意**：
- 识字句是**独立的段落**，不是表格的一部分
- 直接输出完整句子，不需要额外格式
- 如果图片中没有识字句，可以省略此部分

---

### 3. 词组（Phrases）

**标题格式**（二选一）：
- `## 识字9 - 词组`（推荐）
- `## 词组`（简化格式）

**表格格式**：
```markdown
## 识字9 - 词组

| 词组 | 拼音 | 英文 |
|----|----|----|
| 叔叔 | shūshu | uncle |
| 人口 | rénkǒu | population |
| 爸爸 | bàba | dad |
| 妈妈 | māma | mum |
```

**字段说明**：
- `词组`：中文词组（必填）
- `拼音`：拼音（必填，带声调）
- `英文`：英文释义（必填）

**注意**：
- 表格第一行必须有开头的 `|`
- 如果图片中没有词组部分，可以省略

---

### 4. 语法（Grammar）

**标题格式**（二选一）：
- `## 识字9 - 语法`（推荐）
- `## 语法`（简化格式）

**格式一**（推荐，同一行）：
```markdown
## 识字9 - 语法

**规则1** (语气助词"呢"的用法) | Modal particle "呢" used for questions (after nouns/pronouns), marking a pause, or indicating an action in progress.
**规则2** (副词"多"的用法) | Adverb "多" used to ask about degree (多+Adj) or to indicate an approximate number (placed after integers).
```

**格式二**（换行格式，也支持）：
```markdown
## 识字9 - 语法

**规则1** (语气助词"呢"的用法)
| Modal particle "呢" used for questions (after nouns/pronouns), marking a pause, or indicating an action in progress.

**规则2** (副词"多"的用法)
| Adverb "多" used to ask about degree (多+Adj) or to indicate an approximate number (placed after integers).
```

**字段说明**：
- `**规则X**`：规则编号（X 为数字，从 1 开始）
- `(中文说明)`：中文语法说明（必填）
- `| 英文翻译`：英文翻译（必填，用 `|` 分隔或换行）

**注意**：
- 如果图片中没有语法部分，可以省略

---

### 5. 练习（Exercises）

**标题格式**（三选一）：
- `## 识字9 - 练习`（完整格式，推荐）
- `## 练习`（简化格式）
- 直接以 `### 选择题` 开始（仅练习部分时使用）

#### 5.1 填空题格式

```markdown
## 识字9 - 练习

### 填空题

**题目1**: 你_A_妈妈_B_不喜欢喝花茶_C_，教你的老师_D_？
- 答案位置：D
- 填空词：呢

**题目2**: 我_A_写汉字_B_，_C_我父母_D_作饭呢。
- 答案位置：A
- 填空词：在

**题目3**: 他_A_说，昨天_B_他_C_他的同学_D_打了。
- 答案位置：C|D
- 填空词：被
```

**字段说明**：
- `### 填空题`：子类型标题（必填）
- `**题目X**:`：题目编号和内容（必填，X 为数字）
- `- 答案位置：X`：正确答案的位置（单选填一个字母，如 `D`；多选用 `|` 分隔，如 `C|D`）
- `- 填空词：词语`：需要填入的词语

**注意**：
- 题目中的位置标记格式为 `_A_`、`_B_`、`_C_`、`_D_`
- 每个题目之间用空行分隔
- **如果题目有多个正确答案位置**，答案位置格式为 `C|D`（用 `|` 分隔多个位置）

#### 5.2 选择题格式

```markdown
### 选择题

**题目1**: 她___开汽车，不过，开得不太好。
- 答案：D
- 选项：A.能|B.可以|C.想|D.会

**题目2**: 如果我们不懂，老师___我们问她。
- 答案：B
- 选项：A.被|B.让|C.叫|D.说

**题目3**: 三斤花茶三百六十九块，一斤花茶多少钱？
- 答案：B|C
- 选项：A.一百三十三块|B.一百二十三块|C.一百四十二块|D.一百三十二块
```

**字段说明**：
- `### 选择题`：子类型标题（必填）
- `**题目X**:`：题目编号和内容（必填）
- `- 答案：X`：正确答案（单选填一个字母，如 `D`；多选用 `|` 分隔，如 `B|C`）
- `- 选项：A.选|B.选|C.选|D.选`：选项列表（必填，用 `|` 分隔）

**注意**：
- 如果题目是多选题，答案格式为 `B|C`（用 `|` 分隔多个答案）
- 选项之间用 `|` 分隔，不要用换行

#### 5.3 对话格式（如果有）

```markdown
### 对话

**题目1**: 问：你好吗？| 答：我很好，谢谢。
**题目2**: 问：今天天气怎么样？| 答：今天天气很好。
```

## ⚠️ 重要注意事项

1. **标题格式**：
   - ✅ 正确：`## 识字9 - 字表`（使用阿拉伯数字）
   - ❌ 错误：`## 识字九 - 字表`（不要使用中文数字）

2. **表格格式**：
   - ✅ 正确：`| 字 | 拼音 | 英文 | notes |`（有开头的 `|`）
   - ❌ 错误：`字 | 拼音 | 英文 | notes`（没有开头的 `|`）

3. **空行要求**：
   - 标题与内容之间必须有空行
   - 不同部分之间建议有空行分隔

4. **练习格式**：
   - 填空题：使用 `- 答案位置：X` 和 `- 填空词：词语`
     - 单选：`- 答案位置：D`
     - 多选：`- 答案位置：C|D`（多个正确答案位置用 `|` 分隔）
   - 选择题：使用 `- 答案：X` 和 `- 选项：A.选|B.选|C.选|D.选`
     - 单选：`- 答案：D`
     - 多选：`- 答案：B|C`（多个正确答案用 `|` 分隔）

5. **如果图片中没有某个部分**：
   - 可以省略该部分（如没有语法部分，就不输出语法）
   - 但如果有，必须按照格式输出

6. **保持原始内容**：
   - 不要修改图片中的原始内容
   - 只负责格式转换，不负责内容修正
