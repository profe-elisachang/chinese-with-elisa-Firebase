练习 1
选择题：哪句使用了"一...就..."？
1. 从明天开始上课。
2. 一到教室就上课。
3. 早上七点一刻去上课。
4. 他专教留学生。
+ 显示答案
正确答案：2. 一到教室就上课。

"一...就..."表示两个动作紧接着发生。选项2完整地展示了这个结构。
练习 2
排序题：用"一...就..."结构将词语排序。
1. 下课 / 一 / 我 / 就 / 回宿舍
2. 一 / 毕业 / 他 / 工作 / 就
3. 一 / 退休 / 就 / 他们 / 休息
4. 一 / 见面 / 他们 / 打招呼 / 就
+ 显示答案
1. 我一下课就回宿舍。
2. 他一毕业就工作。
3. 他们一退休就休息。
4. 他们一见面就打招呼。

练习 1
选择题：哪句中的"要...了"使用正确？
1. 昨天他要回家了。
2. 商店要关门了。
3. 他以前要当老师了。
4. 我已经要吃饭了。
+ 显示答案
正确答案：2. 商店要关门了。

"要...了"表示即将发生的事。只有选项2符合这个用法。
练习 2
判断正误：根据课文判断，正确的打✓，错误的打✗。
1. 女朋友的生日要到了。( )
2. 王先生要教英语了。( )
3. 林叔叔的祖父母要退休了。( )
4. 这孩子大学要毕业了。( )
+ 显示答案
1. 女朋友的生日要到了。(✓) 课文01：这个星期天是她的生日
2. 王先生要教英语了。(✗) 课文02：他专教...汉语和书法
3. 林叔叔的祖父母要退休了。(✗) 课文09：已经退休很长时间了
4. 这孩子大学要毕业了。(✗) 课文10：大学还没毕业

练习 1
选择题：选择正确的一项。
这些东西太贵了，我______。

1. 买得
2. 买不了
3. 买不到
4. 买得了
+ 显示答案
正确答案：2. 买不了

"V+不+了"表示没能力做某事。这里表示因为太贵而无法购买。
练习 2
转换题：根据课文意思转换句子。
1. 课文07：这些东西我可以买。 → 这些东西我______。
2. 课文08：我们可以喝完五瓶啤酒。 → 五瓶啤酒我们______。
3. 课文10：他能当研究员。 → 研究员他______。
4. 课文09：祖父母不能工作了。 → 祖父母______工作了。
+ 显示答案
1. 这些东西我买得了。
2. 五瓶啤酒我们喝得了。
3. 研究员他当得了。
4. 祖父母工作不了了。
练习 3
填空题：用"V+得/不+了"填空。
1. 自行车坏了，我______教室。
2. 钱不够，我______那么多东西。
3. 他很有经验，______留学生。
4. 我不知道名字，______服务员。
+ 显示答案
1. 自行车坏了，我到不了教室。
2. 钱不够，我买不了那么多东西。
3. 他很有经验，教得了留学生。
4. 我不知道名字，问不了服务员。
练习 4
情景对话：用"V+得/不+了"回答问题。
1. A: 你能记住这些单词吗？(太多)
    B: _______________________________

2. A: 后天你能和我们一起去商店吗？(有事)
    B: _______________________________

3. A: 你会回答这个问题吗？(会)
    B: _______________________________

4. A: 你能吃完这盘菜吗？(太多了)
    B: _______________________________
+ 显示答案
参考答案：
1. B: 我记不了，太多了。
2. B: 我去不了，我有事。
3. B: 我回答得了。
4. B: 我吃不了，太多了。

练习 1
选择题：哪句是反问句？
1. 你上哪儿去啊？
2. 吃饭了吗？
3. 谁不想成绩好啊！
4. 您贵姓？
+ 显示答案
正确答案：3. 谁不想成绩好啊！

反问句用否定的形式表达肯定的意思。这个句子用来强调大家都想成绩好。

1. 他是老师，就是医生。
2. 运动衣好是好，就是太贵。
3. 我想去，就是没时间。

1. 鱼香肉丝好吃是好吃
2. 当研究员好是好
3. 骑自行车快是快
